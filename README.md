This project is an experimental engine designed to let you experience the fourth dimension.
It features a complete rendering pipeline and tools for importing your own models, so you can explore higher-dimensional spaces without worrying about low-level graphics details.

What started as a personal challenge — to build everything from scratch using only a graphics library (and eventually a networking library) — has grown into a platform for experimenting with 4D worlds. Every system, from rendering to interaction, has been hand-coded by me.

The ultimate goal is to create something like a “4D Minecraft”: an interactive environment where you can build, explore, and maybe even solve puzzles in four dimensions. Along the way, I plan to add game mechanics, challenges, and multiplayer networking, all while staying true to the minimal-library challenge.

If you’re curious about what it’s like to think, move, and build in a world beyond three dimensions, this project is a playground for imagination and experimentation.
