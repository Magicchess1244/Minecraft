This project is an experimental engine designed to let you experience the fourth dimension 🌀.
It comes with a fully working rendering pipeline and tools to import your own models 🏗️, so you can dive straight into higher-dimensional exploration without worrying about the low-level graphics details.

What started as a personal challenge 💪 — building everything from scratch using only a graphics library (and later a networking library 🌐) — has become a playground for 4D experimentation. Every system, from rendering to interaction, has been hand-coded by me 🖥️✍️.

The ultimate goal? A “4D Minecraft” 🧱🌀: an interactive world where you can build, explore, and maybe even solve puzzles in four dimensions 🕹️. In the future, I plan to add game mechanics, goals, and multiplayer networking, all while sticking to the minimal-library challenge.

If you’re curious about what it’s like to think, move, and create in a world beyond three dimensions 🌠, this project is your sandbox for imagination and discovery 🎮✨.
